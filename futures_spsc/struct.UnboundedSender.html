The transmission half of the unbounded spsc channel.

This structure can be used to send messages to a receiver, and all messages will be buffered internally.

Each sender implements the Sink trait but also provide an inherent send method to statically handle the case that NotReady never arises.

fn send(&mut self, t: T) -> Result<(), SendError<T>>

Sends a message over this channel to the receiver.

The message t is guaranteed to be sent and will be buffered internally so long as the receiver is still alive. If the receiver has dropped then the message is returned as an UnboundedSendError. If the message was queued then Ok(()) is returned.
